## Publicação rápida
1. Abra no VS Code e teste com Live Server.
2. Copie tudo para `C:\Sites\Polartica` no servidor.
3. Configure IIS (portas 80/443), gere SSL (Certify The Web) e redirecione HTTP→HTTPS.
